# EggsDontFit
Deny spawn eggs from being put in spawners

## Requirements
* Spigot 1.13.2+
* Java 1.8+

### Permissions
| Permission                 | Description   | Default |
| -------------------------- | ------------- | :-----: |
| eggsdontfit.denyspawnerchange | Set to ``true`` to deny player from changing the spawner  | No |
